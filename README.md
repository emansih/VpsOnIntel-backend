## Description

The backend application to interact with Niantic's Visual Positioning System (VPS). Hosted on Vercel at `https://vps-on-intel.vercel.app`


## Project setup

In the root directory, make sure `.env` file is included. 

The following variables are required

```
LIGHTSHIP_USERNAME="xxxx"
LIGHTSHIP_PASSWORD="xxxx"
KV_URL="xxxx"
KV_REST_API_TOKEN="xxx"
KV_REST_API_URL="xxx"

```

To get KV token and URL, follow the integration guide here: https://vercel.com/marketplace/upstash

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
